# 🚀 SkillForge AI

AI-powered career development platform.

## Features

- JWT Authentication
- AI Career Assistant
- AI Resume Analyzer
- AI Career Roadmap Generator
- Learning Progress Tracker
- Admin Analytics Dashboard


## Tech Stack

Frontend:

- React
- Vite
- Tailwind CSS


Backend:

- Node.js
- Express.js
- MongoDB


AI:

- Groq LLM API


## Architecture

React
|
Express API
|
MongoDB
|
AI Engine


## Installation


Clone:

git clone repo-url


Frontend:

cd client

npm install

npm run dev



Backend:

cd server

npm install

npm run dev



## Environment Variables

MONGO_URI=
